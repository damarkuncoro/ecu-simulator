**Proof of the Extract Phase Reduction (HKDF Game 0 → Game 1)**

Here is the detailed, formal proof for the transition from **Game 0** (real HKDF) to **Game 1** (random PRK).

### Game Definitions (Extract Focus)

**Game 0 (Real Extract)**:
For each query $(salt, info, len)$:
$$
PRK := \mathsf{HMAC.Hash}(salt, IKM)
$$
$$
output := \mathsf{Expand}(PRK, info, len)
$$

**Game 1 (Random PRK)**:
For each query $(salt, info, len)$:
$$
PRK \xleftarrow{\$ } \{0,1\}^n \quad \text{(uniform random string of length } n\text{)}
$$
$$
output := \mathsf{Expand}(PRK, info, len)
$$

---

### Theorem (Extract Reduction)

$$
|\Pr[\mathcal{A} \text{ wins Game 0}] - \Pr[\mathcal{A} \text{ wins Game 1}]| \leq \mathsf{Adv}^{\mathsf{PRF}}_{\mathsf{HMAC}}(\mathcal{B})
$$

for some efficient PRF adversary $\mathcal{B}$ whose running time and number of queries are essentially the same as $\mathcal{A}$'s.

### Proof by Reduction

We construct a **PRF distinguisher $\mathcal{B}$** that breaks the Pseudorandom Function security of HMAC using $\mathcal{A}$ as a black box.

#### Construction of $\mathcal{B}$:

1. $\mathcal{B}$ receives a **PRF challenge oracle** $\mathcal{O}_k(\cdot)$, where $k$ is a secret key chosen by the PRF challenger. $\mathcal{O}$ is either:
   - Real: $\mathsf{HMAC}(k, \cdot)$
   - Random: a truly random function $R(\cdot)$

2. $\mathcal{B}$ runs $\mathcal{A}$ as a subroutine:
   - When $\mathcal{A}$ makes an HKDF query $(salt_i, info_i, len_i)$:
     - $\mathcal{B}$ queries its oracle once:  
       $$
       PRK_i \gets \mathcal{O}(salt_i)
       $$
     - Then computes the Expand phase **normally** (using the real Expand algorithm):  
       $$
       output_i \gets \mathsf{Expand}(PRK_i, info_i, len_i)
       $$
     - Returns $output_i$ to $\mathcal{A}$.

3. When $\mathcal{A}$ outputs its guess $b'$ (trying to distinguish the two games), $\mathcal{B}$ outputs the same $b'$ as its guess for whether its oracle $\mathcal{O}$ is real HMAC or random.

### Correctness Analysis

- **Case 1**: The PRF challenger gave $\mathcal{B}$ a **real HMAC oracle**.  
  Then $PRK_i = \mathsf{HMAC}(salt_i, IKM)$ exactly as in **Game 0**.  
  → $\mathcal{A}$'s view is identical to Game 0.

- **Case 2**: The PRF challenger gave $\mathcal{B}$ a **random function**.  
  Then $PRK_i$ is uniformly random (independent of $salt_i$).  
  → $\mathcal{A}$'s view is identical to **Game 1**.

Therefore, $\mathcal{B}$ wins the PRF game exactly when $\mathcal{A}$ distinguishes Game 0 from Game 1. Hence the advantage is preserved:
$$
\mathsf{Adv}^{\text{Game0 vs Game1}}(\mathcal{A}) = \mathsf{Adv}^{\mathsf{PRF}}_{\mathsf{HMAC}}(\mathcal{B})
$$

This is a **tight, direct reduction**.

### Why This Works Mathematically

The Extract phase is **exactly one invocation** of HMAC with:
- Key = $IKM$ (fixed for the session)
- Input = $salt$

This matches the standard PRF definition perfectly:
$$
\mathcal{O}(x) = \mathsf{HMAC}(k, x)
$$

The rest of the protocol (Expand) is deterministic and identical in both games, so it does not affect the distinguishing advantage.
