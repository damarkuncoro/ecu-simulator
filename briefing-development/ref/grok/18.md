**Proof of the Expand Phase Reduction (Game 1 → Game 2)**

Here is the detailed formal proof.

### Game Definitions

**Game 1** (Random PRK + Real Expand):
- Challenger samples $PRK \xleftarrow{\$ } \{0,1\}^n$.
- On query $(salt, info, len)$:
  $$
  \begin{align*}
  T_0 &= \epsilon \\
  T_j &= \mathsf{HMAC}(PRK,\ T_{j-1} \ ||\ info\ ||\ \mathsf{counter}(j)) \quad \text{for } j = 1,2,\dots \\
  output &= \text{first } len \text{ bytes of } T_1 || T_2 || \cdots
  \end{align*}
  $$

**Game 2** (Ideal Random Output):
- On query $(salt, info, len)$:
  $$
  output \xleftarrow{\$ } \{0,1\}^{len} \quad \text{(uniform random string)}
  $$

---

### Theorem

$$
|\Pr[\mathcal{A}\ \text{wins Game 1}] - \Pr[\mathcal{A}\ \text{wins Game 2}]| \ \leq\ q \cdot \mathsf{Adv}^{\mathsf{PRF}}_{\mathsf{HMAC}}(\mathcal{C})
$$

where $q$ is the number of HKDF queries made by $\mathcal{A}$, and $\mathcal{C}$ is a PRF adversary.

### Proof by Reduction

We construct a PRF distinguisher $\mathcal{C}$ that breaks the PRF security of HMAC using $\mathcal{A}$ as a black box.

#### Construction of $\mathcal{C}$:

1. $\mathcal{C}$ receives a PRF oracle $\mathcal{O}$ (either real $\mathsf{HMAC}(k, \cdot)$ or a random function).

2. $\mathcal{C}$ simulates Game 1 for $\mathcal{A}$:
   - For each query $(salt_i, info_i, len_i)$ from $\mathcal{A}$:
     - Sample a fresh random $PRK_i \xleftarrow{\$ } \{0,1\}^n$.
     - Simulate the Expand chain **using the oracle $\mathcal{O}$**:
       $$
       T_0 = \epsilon
       $$
       For $j = 1,2,\dots$ until enough bytes:
       $$
       T_j \gets \mathcal{O}(T_{j-1}\ ||\ info_i\ ||\ \mathsf{counter}(j))
       $$
     - Set $output_i$ = first $len_i$ bytes of the concatenation of all $T_j$.
     - Return $output_i$ to $\mathcal{A}$.

3. When $\mathcal{A}$ outputs guess $b'$, $\mathcal{C}$ outputs the same $b'$.

### Analysis

- **Real HMAC case** ($\mathcal{O}$ is real with key $= PRK_i$):  
  Simulation is **identical to Game 1**.

- **Random function case**:  
  Each $T_j$ is independently random (fresh PRK + domain separation via counter).  
  The concatenated output is statistically close to uniform random → **identical to Game 2**.

### Advantage Bound

Since $\mathcal{A}$ makes $q$ independent queries, and each Expand simulation uses a small number of HMAC calls, the total distinguishing advantage is bounded by a factor of $q$ times the single-call PRF advantage against HMAC.

This completes the **Expand phase reduction**.

---

**Overall HKDF Bound** (combining both reductions):

$$
\text{Adv}^{\text{HKDF}}(\mathcal{A}) \ \leq\ \mathsf{Adv}^{\mathsf{PRF}}_{\mathsf{HMAC}}(\mathcal{B}) + q \cdot \mathsf{Adv}^{\mathsf{PRF}}_{\mathsf{HMAC}}(\mathcal{C})
$$

This is the standard game-hopping proof used in the literature for HKDF.
